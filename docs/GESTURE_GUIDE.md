## System Control Gestures

### ✋ Open Palm
**Action:** Pause current operation

**How to perform:**
- Extend all five fingers
- Keep palm facing the camera
- Hold steady for half a second

**Use cases:**
- Pause video playback
- Stop current operation
- Emergency pause

```
    _ _ _ _
   |       |
   | | | | |
   | | | | |
   |_|_|_|_|
```

---

### ✊ Closed Fist
**Action:** Confirm/Select

**How to perform:**
- Curl all fingers into palm
- Thumb can be inside or outside
- Hold steady

**Use cases:**
- Confirm dialog boxes
- Select items
- Click in cursor mode

```
    _______
   |       |
   | | | | |
   |_|_|_|_|
```

---

### 👍 Thumbs Up
**Action:** Accept/Yes/Continue

**How to perform:**
- Extend thumb upward
- Keep other fingers folded
- Palm can face any direction

**Use cases:**
- Accept prompts
- Confirm actions
- Positive response

```
    _ _ _ _
   | | | | |
   | | | | |
   | | | | |
   _|_|_|_|_
```

---

### 👎 Thumbs Down
**Action:** Reject/No/Cancel

**How to perform:**
- Extend thumb downward
- Keep other fingers folded
- Similar to thumbs up but inverted

**Use cases:**
- Reject prompts
- Cancel operations
- Negative response

```
    _ _ _ _
   | | | | |
   | | | | |
   |_|_|_|_|
      | |
```

---

### ☝️ Pointing
**Action:** Enter Cursor Mode

**How to perform:**
- Extend index finger
- Keep other fingers folded
- Move hand to control cursor

**Use cases:**
- Control mouse cursor
- Point to items
- Precision selection

```
    _ _ _ _
   | | | | |
   | | | | |
   | | |_|_|
   |_|_|_|
```

---

### ✌️ Peace Sign
**Action:** Take Screenshot

**How to perform:**
- Extend index and middle fingers
- Form a V shape
- Keep other fingers folded

**Use cases:**
- Capture full screen
- Quick screenshot

```
    _ _ _ _
   | | | | |
   | | | | |
   | |_|_|_|
   |_|_|_|
```

---

### 🤘 Rock On
**Action:** Toggle Play/Pause

**How to perform:**
- Extend index and pinky fingers
- Fold middle and ring fingers
- Thumb can be extended or folded

**Use cases:**
- Play/pause media
- Quick media control

```
    _ _ _ _
   |_|_|_|_|
      |   |
      |   |
```

---

### 👌 OK Sign
**Action:** Open Settings

**How to perform:**
- Touch thumb tip to index fingertip
- Form a circle (OK shape)
- Keep other fingers extended

**Use cases:**
- Open settings menu
- Access configuration

```
     _ _ _
    /     \
   |   _   |
    \__|__/
```

## Navigation Gestures

### 👈 Swipe Left
**Action:** Previous Workspace/Window

**How to perform:**
- Open palm
- Move hand quickly from right to left
- Keep palm facing camera

**Use cases:**
- Switch to previous workspace
- Go back in browser
- Previous window

---

### 👉 Swipe Right
**Action:** Next Workspace/Window

**How to perform:**
- Open palm
- Move hand quickly from left to right
- Keep palm facing camera

**Use cases:**
- Switch to next workspace
- Go forward in browser
- Next window

---

### 👆 Swipe Up
**Action:** Scroll Up

**How to perform:**
- Open palm
- Move hand quickly upward
- Keep palm facing camera

**Use cases:**
- Scroll up page
- Maximize window

---

### 👇 Swipe Down
**Action:** Scroll Down

**How to perform:**
- Open palm
- Move hand quickly downward
- Keep palm facing camera

**Use cases:**
- Scroll down page
- Minimize window

---

### 🔄 Circular (Clockwise)
**Action:** Refresh/Reload

**How to perform:**
- Extend index finger
- Make circular motion clockwise
- Keep other fingers relaxed

**Use cases:**
- Refresh page
- Reload content
- Update view

---

### 🔃 Circular (Counter-Clockwise)
**Action:** Undo

**How to perform:**
- Extend index finger
- Make circular motion counter-clockwise
- Keep other fingers relaxed

**Use cases:**
- Undo last action
- Reverse operation

---

### 🤏 Pinch In
**Action:** Zoom Out

**How to perform:**
- Extend thumb and index finger
- Move them closer together
- Can be done with motion or held position

**Use cases:**
- Zoom out
- Decrease magnification

---

### 👐 Pinch Out
**Action:** Zoom In

**How to perform:**
- Start with thumb and index finger close
- Move them apart
- Can be done with motion or held position

**Use cases:**
- Zoom in
- Increase magnification

---

---

### 👋 Wave
**Action:** Cancel Operation

**How to perform:**
- Open palm
- Wave side to side
- Similar to waving goodbye

**Use cases:**
- Cancel current operation
- Dismiss notifications

## Tips for Better Recognition

### Lighting
- Ensure good, even lighting
- Avoid backlighting
- Natural light works best

### Position
- Keep hand 1-3 feet from camera
- Center hand in frame
- Avoid extreme angles

### Movement
- Make gestures deliberately
- Hold gestures for at least 0.5 seconds
- Avoid rapid, jerky movements

### Practice
- Use calibration mode to train system
- Practice each gesture 5-10 times
- System learns your specific hand shape

## Troubleshooting

### Gesture Not Recognized
1. Check lighting conditions
2. Ensure hand is fully visible
3. Try adjusting confidence threshold
4. Re-run calibration

### False Positives
1. Increase confidence threshold
2. Make gestures more deliberately
3. Ensure clear background

### Delayed Response
1. Check CPU usage
2. Close unnecessary applications
3. Reduce camera resolution
4. Use lighter AI model

## Custom Gestures

You can train custom gestures:

```bash
# Collect data for custom gesture
python scripts/collect_gesture_data.py --gesture my_custom_gesture --samples 100

# Retrain model with new gesture
python scripts/train_gesture_model.py
```

## Gesture Confidence Levels

| Level | Range | Description |
|-------|-------|-------------|
| High | 0.85+ | Very reliable |
| Medium | 0.70-0.85 | Good reliability |
| Low | 0.50-0.70 | May need adjustment |

Adjust thresholds in `config/default_config.yaml`:

```yaml
gesture:
  confidence_threshold: 0.75  # Adjust this value
```
