## Advanced Usage

### Combining Commands

You can combine multiple commands in sequence:

```
"Hey System, open Chrome, search for Python tutorials"
```

### Contextual Commands

Some commands work based on context:

```
# When a file is selected
"Send to MacBook"  # Sends selected file

# When media is playing
"Next"  # Next track

# In a browser
"Go back"  # Browser back
```

### Natural Language

HGVCS understands natural variations:

```
"Open Google Chrome"
"Launch Chrome"
"Start Chrome browser"
```

All of these will open Chrome.

## Tips for Better Recognition

### Speaking
- Speak clearly and at normal pace
- Use natural language
- Avoid background noise
- Keep microphone 6-12 inches away

### Command Structure
- Start with wake word: "Hey System"
- Pause briefly after wake word
- Speak command clearly
- Wait for confirmation beep

### Common Mistakes

❌ **Don't:**
- Speak too fast
- Mumble or whisper
- Use very long commands
- Speak while system is processing

✅ **Do:**
- Speak naturally
- Use simple, clear commands
- Wait for processing to complete
- Repeat if not understood

## Custom Commands

You can add custom commands in `config/default_config.yaml`:

```yaml
voice:
  commands:
    my_custom_command:
      patterns:
        - "custom phrase"
        - "alternative phrase"
      action: "custom_action"
      description: "What this command does"
```

Then implement the action in `src/control/system_controller.py`.

## Troubleshooting

### Command Not Recognized
1. Check microphone permissions
2. Adjust silence threshold
3. Speak more clearly
4. Try simpler phrasing

### False Activations
1. Change wake word
2. Adjust wake word sensitivity
3. Use push-to-talk mode

### Delayed Response
1. Check CPU usage
2. Use lighter Whisper model
3. Reduce audio buffer size

## Configuration

Adjust voice settings in `config/default_config.yaml`:

```yaml
voice:
  wake_word: "Hey System"
  confidence_threshold: 0.7
  silence_threshold: 0.02
  model_size: "base"  # tiny, base, small, medium, large
```

### Model Sizes

| Model | Size | Speed | Accuracy |
|-------|------|-------|----------|
| tiny | 39 MB | Fastest | Basic |
| base | 74 MB | Fast | Good |
| small | 244 MB | Medium | Better |
| medium | 769 MB | Slow | Best |
| large | 1550 MB | Slowest | Excellent |
